# Описание git

## Рабочие зоны git

1. project's working directory(pwd)
1. staging area(sa) in file .git/index
1. repository(rp) in directory .git 
1. origin(or)
1. upstream(up)

## Инициализация локальная
```
mkdir <directory>
cd <directory>
git init
```
